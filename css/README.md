# PashyamiSoul — Personal Identity & Portfolio Website

A modern multi-page personal portfolio and identity website built with HTML, CSS, GitHub, and Vercel.

Live Site:
https://pashyamisoul-site.vercel.app

---

## Overview

PashyamiSoul is a minimalist personal website focused on:

- Personal branding
- Professional portfolio presentation
- Recruiter accessibility
- Responsive design
- Smooth user experience
- Clean editorial aesthetics

The project evolved from a static portfolio into a complete identity-focused website with a dedicated bio landing page and portfolio experience.

---

## Features

### Bio Landing Page
- Personal introduction
- Animated typing effect
- Social media links
- Cinematic UI styling
- Responsive mobile design
- Smooth entrance animations
- Ambient background glow effects

### Portfolio Experience
- Professional experience timeline
- Company logo integration
- External company redirects
- Recruiter-friendly resume download
- Multi-page navigation
- Responsive layout
